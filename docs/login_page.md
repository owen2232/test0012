# Login Page

## Purpose
To allow existing users to log in to their Smart Farm account.

## Content
- **Form Fields:**
    - Username/Email (required)
    - Password (required)
- **Remember Me (Optional):**
    - Checkbox for persistent login.
- **Call to Action:**
    - "Login" button.
- **Links:**
    - "Forgot Password?" link.
    - "Don't have an account? Sign Up" link.
